Command Line Interface

pwd

Print working directory

"Wo bin ich?"

ls

List

"Was habe ich in diesem Ordner?"

ls -1

List in einer Spalte

"Was habe ich in diesem Ordner?"

ls -l

List aber in long format

"Was habe ich in diesem Ordner?"

ls -a

List aber mit versteckten Daten

"Was habe ich in diesem Ordner?"

clear

Terminal leeren

"Lass uns neu starten"

cd

Change directory

"Ich will in einem anderen Ordner"

cd ..

Change directory zurück

"Ich will zurück in obenen Ordner"

cd ~

Oberste Ordner

"Ich will zurück zum obersten Ordner"

touch

Datei erstellen

"Ich erstelle ein neu Datei"

mkdir

Make directory

"Ich will ein neuer Ordner machen"

mv

Move

"Ich will ein Datei verschieben"

cp

Copy

"Ich will ein Datei kopieren"

help

Help

"Ich brauche Hilfe mit den Befehlen"

exit

Exit

"Ich will den Terminal beenden"

GIT commands

git init

Initialize git

"Ich will einen neuen Repository erstellen"

git add

Add to staging area

"Ich will Änderungen vorschlagen"

git commit

Commit changes

"Ich will Änderungen bestätigen"

git push

Push changes

"Ich will Änderungen hochladen"

git remote

Connect to remote repository

"Ich will mit einem Repository verbinden"

git log

Show change logs

"Ich will eine Liste alle Änderungen sehen"

git pull

Pull changes from remote repository

"Ich will lokales Repository mit dem remote Repository aktualisieren"

git clone

Clone repository

"Ich will ein Repository auschecken"

git branch

Show branch

"Ich will mein Branch sehen oder wählen"

git checkout

Create and/or change to another branch

"Ich will in einem anderen Branch arbeiten"
